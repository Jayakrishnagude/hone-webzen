# Hey One Webstudio

A premium, human-crafted, production-quality website for a modern web agency and digital experience studio.

## Tech Stack
- **Next.js 14** (App Router)
- **Tailwind CSS** (Styling)
- **Framer Motion** (Animations)
- **React Three Fiber** (3D Visuals)
- **Lucide React** (Icons)

## Features
- **Cinematic 3D Hero Section**: Immersive 3D scene with floating panels and a glowing grid.
- **Premium Glassmorphism Design**: High-end visual hierarchy with deep space themes.
- **Custom Cursor**: Interactive glowing mouse pointer.
- **Rotating Semi-Circle Interface**: Futuristic control arc that reacts to scroll.
- **SEO Optimized**: Semantic HTML, meta tags, and high-performance architecture.
- **Responsive**: Flawless experience across all devices.

## Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`
